# Escala do Culto - Publicacao no GitHub Pages

## Arquivos

- `index.html`: arquivo principal para publicar no GitHub Pages.
- `escala-culto-atibaia.html`: copia com o nome antigo, caso voce queira manter o link antigo localmente.

## Como publicar

1. Crie uma conta em https://github.com.
2. Crie um repositorio novo, por exemplo `escala-culto-atibaia`.
3. Envie o arquivo `index.html` para a raiz do repositorio.
4. No repositorio, entre em `Settings`.
5. Va em `Pages`.
6. Em `Build and deployment`, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
7. Clique em `Save`.

Depois de alguns minutos, o site deve ficar disponivel em:

`https://SEU-USUARIO.github.io/escala-culto-atibaia/`

## Backup

A aplicacao salva os dados no navegador de cada aparelho.

Use:

- `Exportar backup`: baixa um arquivo `.json` com todos os dados.
- `Importar backup`: carrega esse arquivo em outro navegador ou computador.

Guarde o arquivo de backup em um local seguro.
